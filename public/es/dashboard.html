Ver el Tutorial de Panel de Control

Lo que puedes hacer:

  • Establecer horarios comerciales globales para todos los usuarios
  • Personalizar horarios para equipos individuales
  • Crear excepciones (ej. días festivos, paradas estacionales)
  • Sincronizar disponibilidad con programación y despacho de trabajos
  • Asegurar que los trabajos solo se programen durante horarios válidos

Consejos para el éxito

  • Establece horarios de trabajo realistas para evitar citas perdidas
  • Usa disponibilidad de equipos para escalonar equipos y reducir tiempo inactivo
  • Usa excepciones para bloquear cierres estacionales o paradas específicas
  • Asegúrate de que los horarios de trabajo se alineen con tus horas disponibles para evitar conflictos
