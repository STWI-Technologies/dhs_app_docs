Crews Management

Overview

Crews represent teams of users that work together on jobs and appointments. Instead of assigning individual technicians to every appointment or estimate, you can group users into a crew and assign the whole crew at once. Each crew has a name, a color for visual identification, and a designated Crew Lead who is the primary point of contact for that team.

From this section you can:

  • Create new crews with a name, color, description, and members.
  • Edit existing crews to change their details, manage members, or reassign the crew lead.
  • Delete crews that are no longer needed (only if not currently assigned to jobs or estimates).
  • View availability for each crew, and jump directly to the Availability page to configure their schedule.
  • View customer feedback and ratings for each crew.
  • Search and filter crews by name or by assigned users.

Important: A crew can only be deleted if it is not currently assigned to any job or estimate. If a crew is in use, the delete button will be disabled and a tooltip will read: "Cannot delete crew assigned to jobs or estimates."

Crews List View

The Crews List View is the main view for managing all your crews. To access it, navigate to Menu > Crews.

At the top of the page you will see:

  • A Search bar with the placeholder "Search crews" — type a keyword to filter crews by name.
  • A Users filter — a multi-select dropdown that lets you filter the list to show only crews that include specific users as members.
  • A New Crew button — click to open the create crew form.

The list uses server-side pagination. By default, 10 crews are shown per page. You can change the page size to 10, 25, or 50 per page. To create a new crew:

Step 1: Click the New Crew button at the top of the Crews List View. A panel will slide open from the right side with the title Add New Crew.

        

Step 2: Fill in the crew details.

Editing a Crew

To edit an existing crew, click the Edit (pencil icon) button on the crew's row, or click the crew name in the table. A panel will slide open from the right side with the title Edit Crew.

The edit panel has the same fields as the create form (Crew Name, Color, Description) plus additional sections:

Members Section (Edit Mode)

In edit mode, the members section changes from a simple multi-select to a members table that gives you more control:

Column

What It Shows

User

The member's avatar and full name. If the user's account has been disabled, a "(Disabled)" label appears next to their name.

Crew Lead

A radio button. Only one member can be the crew lead at a time. Click the radio button next to a different member to reassign the lead role.

Actions

A delete button (trash icon) to remove this member from the crew. If there is only one member left, the delete button is disabled with the tooltip: "Cannot delete last crew member."

To add a new member, use the Add Crew Member dropdown that appears to the right of the members table. Select a user from the dropdown to add them to the crew.

Note: Disabled users are shown in the members table so you can see who was previously on the crew, but they cannot be set as crew lead and are excluded from the active member count shown in the Crews List View.

Availability Section (Edit Mode)

Below the members table, the edit panel shows the crew's current availability schedule as a read-only summary:

  • General Availability — Lists the crew's enabled work days with their time ranges, each marked with a green dot (for example, "Monday, 9:00 AM - 5:00 PM").
  • Exceptions and Special Conditions — Lists any active exceptions (date ranges, months, or seasons), each marked with an orange dot.

If no availability has been configured, it shows: "No availability data."

Click View Availability Settings to navigate to the full Availability page with this crew pre-selected in Per Crew mode.

Customer Feedback Section (Edit Mode)

Below the availability section, the edit panel shows the crew's customer feedback:

  • Displays the crew's average star rating (1 to 5 stars).
  • Shows feedback grouped by crew member.
  • Displays job-specific feedback with the latest description and average rating.

If the crew has no feedback, this section is hidden.

Saving Changes

After making your changes, click Apply Changes at the bottom of the panel. This button is only enabled if you have actually modified something — if no changes have been detected, the button stays disabled.

The button shows "Updating Crew..." while saving. On success, you will see: "Crew successfully updated." On failure: "Failed To Update Crew."

Click Cancel to close the panel without saving any changes.

Deleting a Crew

To delete a crew:

Step 1: Click the Delete (trash icon) button on the crew's row in the list.

Step 2: A confirmation dialog appears with the title Delete and the message: "Are you sure you want to delete this crew?"

Step 3: Click Delete to confirm, or Cancel to go back.

Warning: Deletion is permanent. There is no archive or restore for crews. Once a crew is deleted, it is gone. However, historical records that referenced this crew (such as past jobs or appointments) will retain the crew information.

Note: If the crew is currently assigned to active jobs or estimates, the delete button is disabled and grayed out. The tooltip reads: "Cannot delete crew assigned to jobs or estimates." You must first remove the crew from those jobs or estimates before you can delete it.

Searching and Filtering

Search

The search bar at the top of the Crews List View filters crews by name. Type any keyword and the list updates automatically (with a slight delay for performance). The search is not case-sensitive.

Filter by Users

The Users filter is a multi-select dropdown that shows all users in your account. Select one or more users to see only crews that include those users as members. This is useful when you need to find which crews a specific person belongs to.

Cross-Platform Usage

Crews are used throughout the DHS platform in several important ways:

Appointments

When creating or editing an appointment, you can assign a crew to handle the work. The appointment form includes a crew selection dropdown where you can:

  • Select an existing crew from the dropdown (crews are shown with their color indicator).
  • Quick-add a new crew if the one you need doesn't exist yet.

The system checks the assigned crew's availability when scheduling the appointment. If the appointment falls outside the crew's available hours or during an exception period, the system will flag it.

Jobs

Crews can be assigned to jobs. The system tracks which crews are assigned to active jobs, which is what determines whether a crew can be deleted. The platform can also detect schedule conflicts — if a crew is already assigned to a job during a given time range, the system can warn you before double-booking.

Estimates

When creating an estimate, you can assign a crew. If the estimate is linked to an existing appointment or job, the crew may be auto-filled from that related record.

Availability

Each crew can have its own custom availability schedule (work days, time slots, and exceptions) separate from the global business availability. See the Availability Management section of this manual for full details on configuring per-crew schedules.

Dashboard and Reports

Crew data feeds into several reporting areas:

  • Crew ratings and performance — Average customer ratings per crew.
  • Crew satisfaction metrics — Trends in customer feedback.
  • Crew-wise complaint distribution — Which crews receive the most complaints.
