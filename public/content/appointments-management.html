Appointments Management

Overview

The Appointments Management section is where you schedule, track, and manage all field service appointments for your clients. An appointment represents a single scheduled visit to a client's property, assigned to a specific crew at a specific date and time.

From this section you can:

  • Create new appointments with a 4-step wizard that walks you through details, scheduling, services, and review.
  • Edit existing appointments to update details, reschedule, or change service items.
  • Track status as appointments progress from Draft through Scheduled, Confirmed, En Route, Started, and finally Completed.
  • Send to client to notify clients about their upcoming appointment with a summary of services and billing.
  • Convert appointments into estimates, jobs, or invoices for seamless workflow continuity.
  • Archive appointments you no longer need in your active view.
  • Restore archived appointments back to active status.
  • Permanently delete archived appointments that have no linked records.
  • Use the Scheduler to see appointments on a visual calendar with day, week, month, and list views.
  • Communicate with clients via Chat, add internal Notes, and manage Attachments from the sidebar.

Important: An appointment can only be permanently deleted if it is not linked to any job, estimate, invoice, or timesheet entry. If any linked records exist, you must first remove those links before deletion is possible.

Appointments List View

The Appointments List View is the main view for managing all your appointments. To access it, navigate to Menu > Appointments.

When you open the Appointments List View, you will see a table listing your appointments. Each row represents one appointment and displays the following information:

Column

What It Means

Name

The appointment name. If the appointment is marked as High Priority, a priority icon appears next to the name. If it is a Callback appointment (a follow-up visit), a callback icon appears as well. Click the name to open the appointment's Detail Page. This column is sortable.

Client

The client associated with this appointment. Displays the client's avatar (or initials) and company name.

Scheduled For

The scheduled date, time range (start – end), and duration of the appointment (e.g., Feb 20, 2026 9:00 AM – 11:00 AM • 2h). If no date has been set, it shows "Not Scheduled". This column is sortable.

Crew

The crew assigned to this appointment. Displays a color dot matching the crew's assigned color, followed by the crew name.

Status

The current status of the appointment, displayed as a color-coded tag. If the appointment has a chat with unread messages, a chat icon with the unread count appears next to the status. Clicking that icon takes you to the Inbox. This column is sortable.

Actions

Quick-action buttons: an Edit icon (pencil) to open the Detail Page, and a More Actions menu (three dots) for additional options like Send to Client, View on Scheduler, View on Map, Cancel, Archive, Restore, or Delete.

Active vs. Archived Appointments

At the top of the list, a segmented control lets you switch between two views:

Tab

What It Shows

Active

Appointments that are not archived. These are your current, in-progress, or upcoming appointments.

Archived

Appointments that have been archived. From this tab you can Restore an appointment back to Active, or permanently Delete it (if it has no linked records).

Appointment Statuses

Each appointment has one of the following statuses, displayed as color-coded tags throughout the system:

Status

Description

Draft

The appointment has been created but is not yet fully scheduled. It can still be edited freely. A draft appointment can be saved even without a date/time.

Scheduled

The appointment has a date, time, and crew assigned and is ready for the crew to work on. This is also the default status when restoring an archived appointment.

Confirmed

The client has confirmed the appointment. The appointment is still editable.

En Route

The crew is traveling to the client's location. The appointment is still editable.

Started

The crew has arrived and work is in progress. The appointment is still editable and the timer can be used to track work time.

Completed

All work is finished. The appointment becomes read-only and cannot be edited. It moves to the Archived tab.

Canceled

The appointment was canceled before completion. It becomes read-only and moves to the Archived tab.

No-Show

The client did not show up for the appointment. It becomes read-only and moves to the Archived tab.

Note: Once an appointment is set to Completed, Canceled, or No-Show, it cannot be edited. You can only Restore it (which changes its status back to Scheduled) or Delete it (permanent, if no linked records exist).

1. Creating a New Appointment

To create a new appointment, click the "+ Add Appointment" button at the top-right of the Appointments List View. A side-panel drawer will open with a 4-step wizard.

Step 1: Details

Fill in the basic appointment information:

Field

Required

Description

Appointment Name

Yes

A descriptive name for the appointment (e.g., "Annual HVAC Inspection", "Furnace Repair Follow-up").

Client Account

Yes

Select the client from a searchable dropdown. You can also use Quick Add to create a new client on the fly.

Property

Yes

Select the property where the work will take place. This dropdown auto-populates based on the selected client's properties. You can also Quick Add a new property.

Crew

Yes

Choose the crew that will perform the work. The crew selection affects which dates/times are available in the next step. You can also Quick Add a new crew.

Priority

No

Check this box to flag the appointment as High Priority. A priority icon will appear next to the appointment name throughout the system.

Callback

No

Check this box if this is a follow-up/callback appointment. When checked, a Job dropdown appears so you can link this callback to the original job.

Job

Only if Callback is checked

Select the job this callback appointment relates to. This field only appears when the Callback checkbox is enabled.

After filling in all required fields, click "Next Step" to proceed.

Step 2: Schedule

Choose when the appointment will take place:

Field

Required

Description

Date

Yes

Select the appointment date. The date picker highlights available dates based on the selected crew's availability. Dates where the crew is unavailable are grayed out. Availability is checked within a 90-day window from today.

Time

Yes

Select the start time from the available time slots. Time slots are shown in 30-minute intervals during the crew's business hours (typically 9:00 AM – 5:00 PM).

Duration

Yes

Select how long the appointment will last. Options range from 15 minutes to 8 hours. The default is 1 hour (60 minutes). The system calculates the end time automatically.

Override Availability

No

Toggle this on to bypass crew availability restrictions. When enabled, all dates and all 24-hour time slots become available, allowing you to schedule outside normal business hours.

Schedule Conflicts: If the selected date/time overlaps with another appointment or job already assigned to the same crew, a Schedule Conflict Warning will appear. You can still proceed by clicking "Schedule Anyway" (the button turns amber to alert you), or go back to choose a different time.

After scheduling, click "Next Step". You can also click "Save as Draft" at any point during Steps 2, 3, or 4 to save the appointment in Draft status without completing all steps.

Step 3: Services

Add the services that will be performed during this appointment:

  • Click the service dropdown to search and select from your available labor-type services.

  • For each service added, you can configure:
  • Hours and Minutes — How long this particular service will take.
  • Unit Retail Price — The hourly rate (auto-populated from the service definition but can be adjusted).
  • Total Cost — Calculated automatically based on hours × rate.
  • Duplicate services are not allowed. If a service is already added, it will not appear in the dropdown.

Note: Adding services is optional. You can create an appointment without any services and add them later from the Detail Page.

Click "Next Step" to proceed to the review.

Step 4: Review

Review all the appointment details before finalizing:

  • Client information — Name and contact details.
  • Property address — Where the work will take place.
  • Crew details — The assigned crew and their color.
  • Date and time — The scheduled date, start time, and duration.
  • Services list — All services with hours and pricing.

If a schedule conflict exists, a warning banner will appear on this page as well.

At the bottom you have three options:

Button

What It Does

Save as Draft

Saves the appointment with Draft status. The appointment will appear in your list but will not be considered "scheduled" yet.

Cancel

Closes the modal without saving. All entered data is lost.

Create Appointment

Creates the appointment with Scheduled status (requires all details and schedule to be complete). If there is a schedule conflict, this button shows as "Schedule Anyway" in amber.

2. Appointment Detail Page

Click any appointment row in the List View (or click the appointment name) to open its Detail Page. The Detail Page is divided into a main content area and a right sidebar.

Header

The header shows:

  • Title: "Appointment: [name]" with priority and callback icons if applicable.
  • Status Dropdown: A dropdown to quickly change the appointment's status to any of the 8 available statuses.
  • Start Timer: A button to start tracking work time for this appointment (only visible for non-archived appointments).
  • More Actions (⋮): A menu with additional actions (see section 7 below).

Tab 1: Overview

The Overview tab displays all core appointment information in a card layout:

Field

Description

Title

The appointment name.

Date Created

When the appointment was first created.

Property

The service property address. A "Show Map" link opens the property on the Map view.

Crew

The assigned crew with their color indicator.

Time

The scheduled time range with timezone (e.g., "9:00 AM – 11:00 AM EST • 2h").

Date

The scheduled date (e.g., "Thursday, February 20, 2026").

Callback for

If the appointment is a callback, a link to the related job appears here.

An Edit button at the top-right of the overview card opens the appointment edit modal (same 4-step wizard). This button is disabled for Completed, Canceled, or No-Show appointments.

Below the overview card you will find:

  • Visits Section: If the appointment has related follow-up visits, they are listed here.

  • Services Section: A table listing all service items with columns for service name/description, hours, minutes, rate, and total cost. You can add services, edit existing service hours/pricing  or delete individual service items. Editing is disabled for archived appointments.

  • Bill Section: Shows the billing summary including subtotal, tax, discount, and total. You can also manage alternate billing addresses and "Bill to" client settings here.

Tab 2: Related Records

The Related Records tab shows any estimates, jobs, or invoices linked to this appointment, and lets you create new ones:

Section

Description

Estimates

Shows linked estimates, if any. Click "Create Estimate" to generate a new estimate from the appointment's data. The system copies the client, property, crew, and service items into the new estimate and navigates you to it.

Jobs

Shows linked jobs, if any. Click "Create Job" to convert the appointment into a job. A modal appears to set the job's start/end dates and priority.

Invoices

Shows linked invoices, if any. Click "Create Invoice" to generate an invoice from the appointment's data. A confirmation dialog appears before creating.

Note: If the appointment is not yet Completed, an info banner appears: "Additional related records available after completion." You can still create related records before completion, but the banner serves as a reminder.

Sidebar

The right sidebar provides client information and communication tools:

Client Info Section:

  • Client avatar, company name, and type (Residential / Business).
  • Phone number, email address, and billing address.
  • An Edit button to modify the client's billing details (alternate billing address).
  • A Bill to Client button to assign a different client for billing purposes (limited to 1 bill-to client per appointment).

Bill to Client Section (appears only if a bill-to client has been assigned):

  • Shows the bill-to client's name, phone, email, and billing address.
  • A Change popover to switch to a different bill-to client.
  • A Remove button (trash icon) to remove the bill-to client with a confirmation dialog.

Communication Tabs:

Tab

Description

Chat

Client-facing messaging. Send and receive messages with the client. Supports file attachments and message threading. Unread messages are indicated with a badge count in the List View.

Notes

Internal notes visible only to your team (not sent to the client). Add timestamped notes with optional file attachments (Word, PDF, PPT, TXT, images). Individual notes can be deleted.

Attachments

View and manage all files attached to this appointment. Files are grouped by type with appropriate icons. You can download or preview attached files.

3. Editing an Appointment

You can edit an appointment from several places:

  1. From the List View: Click the pencil (edit) icon in the Actions column, or select "Edit

  1. From the Detail Page: Click the "Edit" button on the Overview card. This opens the same 4-step wizard used during creation, pre-filled with the appointment's current data.

In Edit mode, all four steps (Details, Schedule, Services, Review) are accessible and pre-marked as completed. You can jump to any step to make changes.

Cannot Edit: Appointments with status Completed, Canceled, or No-Show cannot be edited. The Edit button will be disabled and show a tooltip: "This appointment cannot be edited."

Quick-edit:On the Overview tab of the Detail Page, you can also edit service items directly without opening the full edit wizard. Click on a service's hours or minutes fields to modify them inline. Changes are saved in batch.

4. Archiving Appointments

Archiving removes an appointment from the Active list without permanently deleting it. This is useful when an appointment is no longer needed in your day-to-day view but you want to keep the record.

To archive an appointment:

  1. From the List View or Detail Page, click the More Actions menu (⋮).
  2. Select "Archive".

  1. A confirmation dialog will ask: "Are you sure you want to archive this appointment?"
  2. Click "Archive" to confirm.

The appointment will move from the Active tab to the Archived tab.

4.1 Restoring an Archived Appointment

To restore an archived appointment back to active status:

  1. Switch to the Archived tab in the List View.
  2. Find the appointment and click the More Actions menu (⋮).
  3. Select "Restore".
  4. A confirmation dialog will ask: "Are you sure you want to restore this appointment?"
  5. Click "Restore" to confirm.

The appointment's status will be changed to Scheduled and it will reappear in the Active tab.

4.2 Permanently Deleting an Appointment

Permanent deletion is only available for archived appointments. To delete:

  1. Switch to the Archived tab in the List View.
  2. Find the appointment and click the More Actions menu (⋮).
  3. Select "Delete".
  4. A confirmation dialog will ask: "Are you sure you want to permanently delete this appointment?"
  5. Click "Delete" to confirm.

Deletion Restrictions: An appointment cannot be deleted if it has any of the following linked records:

  • A linked Job
  • A linked Estimate
  • A linked Invoice
  • Associated Timesheet Entries

If deletion is blocked, the Delete button will be disabled and a tooltip will explain the reason (e.g., "This appointment cannot be deleted. It has a linked job.").

5. Sending an Appointment to a Client

You can send an appointment notification to the client via the "Send to Client" feature. This sends an email to the client with the appointment details, services, and billing summary.

To send an appointment to a client:

  1. From the List View or Detail Page, click the More Actions menu (⋮).
  2. Select "Send to Client".
  3. A side-panel drawer opens showing a preview of what the client will receive:
  • Appointment Details — Client info, scheduled date/time.
  • Services — All service line items with descriptions and pricing.
  • Billing Summary — Subtotal, tax, discount, required deposit, and total.
  1. Click "Send to Client" to send, or "Cancel" to close without sending.

Requirements: To send an appointment to a client, the appointment must have a scheduled date/time and a crew assigned. Draft appointments without these details will have the "Send to Client" option disabled.

6. Converting Appointments to Other Records

From the Related Records tab on the Appointment Detail Page, you can convert an appointment into other record types. This streamlines your workflow by carrying over the client, property, crew, and service information.

6.1 Creating an Estimate from an Appointment

  1. Open the appointment's Detail Page.
  2. Go to the Related Records tab.
  3. In the Estimates section, click “Convert to Estimate".

  1. The system creates a new estimate with the appointment's client, property, crew, and services pre-populated.
  2. You are automatically navigated to the new estimate's Detail Page.

6.2 Creating a Job from an Appointment

  1. Open the appointment's Detail Page.
  2. Go to the Related Records tab.
  3. In the Jobs section, click "Convert to Job".

  1. A modal appears where you can set the job's start date, end date, and mark it as priority.
  2. Click "Create Job" to finalize. The new job will be linked to the appointment.

6.3 Creating an Invoice from an Appointment

  1. Open the appointment's Detail Page.
  2. Go to the Related Records tab.
  3. In the Invoices section, click "Convert to Invoice".

  1. A confirmation dialog asks: "Are you sure you want to convert this appointment to invoice?"
  2. Click "Convert" to confirm. The system creates a new invoice with the appointment's data and navigates you to it.

Note: Each appointment can have its own set of linked records. Once an estimate, job, or invoice is created from an appointment, it appears in the Related Records tab with a link to navigate directly to it.

8. Time Tracking

Each active appointment has a "Start Timer" button in the Detail Page header. Use this to track the time spent on an appointment:

  • Click "Start Timer" to begin tracking work time. Clicking 'Start Time' will automatically change the status to 'Started'.

  • The timer runs in the background as the crew works.

  • Timer data is linked to timesheets for payroll and reporting purposes.
  • The timer is not available for archived appointments (Completed, Canceled, No-Show).

9. The Scheduler (Calendar View)

In addition to the List View, appointments can be viewed on a visual calendar. To access it, navigate to Menu > Scheduler.

The Scheduler offers the following views:

View

Description

Day View

Shows all appointments for a single day with time slots. You can click on a time slot to create a new appointment with that date/time pre-filled.

Week View

A custom week grid showing appointments across a 7-day period. Supports drag-and-drop to reschedule appointments.

Month View

A traditional month calendar showing appointment counts per day.

List / Agenda View

A list-style view showing appointments in chronological order, similar to an agenda.

Scheduler Features:

  • Crew Filtering: Toggle specific crews on/off to filter which appointments appear.

  • Crew Colors: Each crew has an assigned color. Appointments are displayed in their crew's color for quick visual identification.
  • Unavailable Dates: Dates with no crew availability are visually grayed out.

  • Drag-and-Drop: In the custom Week View, you can drag appointments to a new time slot to reschedule them.

  • Legend: A legend shows crew colors and event types for reference.
  • Date Navigation: Use forward/back arrows to navigate between days, weeks, or months.
  • Override Availability: Toggle to show all dates regardless of crew availability.

10. Searching and Filtering

Quick Search

Use the search bar at the top of the List View to search appointments by keyword. The search filters appointments by name, client name, and other text fields. Results update in real time as you type.

Status Filter

Click the "Status" filter button to open a dropdown with all 8 appointment statuses as checkboxes. You can select one or more statuses to show only appointments matching those statuses.

  • Each status is displayed with its color-coded tag for easy identification.
  • A badge on the filter button shows how many statuses are currently selected.
  • Click "Reset Filter" to clear all status filters and show all appointments.

  • Deselecting all checkboxes has the same effect as resetting — it shows all appointments.

Sorting

The following columns can be sorted in ascending or descending order by clicking the column header:

  • Name
  • Scheduled For (date/time)
  • Status

Pagination

The list uses server-side pagination. You can configure the number of appointments shown per page (10, 25, or 50) using the page size selector at the bottom. Navigate between pages using the pagination controls.
