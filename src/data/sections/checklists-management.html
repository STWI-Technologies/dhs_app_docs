Checklists Management[h][i]

Overview

The Checklists Management section is where you create and manage reusable checklist templates for your business. Checklists help ensure that jobs are completed thoroughly by defining a set of items that must be reviewed or verified before a job can be marked as complete.

Checklists work as templates — you create a checklist once, and then assign it to as many jobs or estimates as needed. Each job gets its own copy of the checklist, so changes made during a job do not affect the original template.

From this section you can:

  • Create new checklist templates with a title, description, and a set of items to verify.
  • Edit existing checklists to add, remove, or reorder items.
  • Duplicate an existing checklist to use it as a starting point for a new one.
  • Archive checklists you no longer need (without losing historical data).
  • Restore archived checklists if you need them again.
  • Permanently delete archived checklists that are no longer needed.
  • Search to quickly find a specific checklist.

Important: A checklist can only be permanently deleted if it is not currently assigned to any job. If a checklist is in use, you must first remove it from all associated jobs before deletion is possible.

Checklists List View

The Checklists List View is the main view for managing all your checklist templates. To access it, navigate to Menu > Checklists.

When you open the Checklists List View, you will see a table listing all your active checklists. Each row represents one checklist and displays the following information:

Column

What It Means

Date Created

The date the checklist was originally created (e.g., "Jan 1, 2026").

Title

The name of the checklist, along with its description (if one was provided). The description is shown as smaller text below the name.

Options

Quick-action buttons to edit, duplicate, or archive/delete the checklist.

Clicking anywhere on a row will open that checklist for viewing and editing its items.

Active vs. Archived Checklists

At the top of the list you will see two tabs:

  • Active — Shows all checklists that are currently available for use. This is the default view. Active checklists can be assigned to new jobs and estimates.
  • Archived — Shows checklists that have been archived (temporarily disabled). Archived checklists do not appear in the dropdown when creating or editing jobs, but any jobs that already have them assigned will continue to work normally.

Sorting the List

You can sort the checklists by clicking on the column headers for Date Created or Title. Click once to sort in ascending order, and click again to sort in descending order.

Searching

Use the search bar to the right of the tabs to find checklists by name or description. Type your search term and the results will update automatically.

1. Creating a New Checklist

To create a new checklist template, follow these steps:

Step 1: Open the New Checklist Form

Click the "New Checklist" button (blue) located in the top-right corner of the Checklists List View. A panel will slide open from the right side of the screen.

Tip: If the list is empty (no checklists have been created yet), you will see a "Create Your First Checklist" button in the center of the page.

Step 2: Enter the Checklist Details

Field

Required?

Description

Checklist Title

Yes

Enter a clear, descriptive name for the checklist (e.g., "HVAC Installation Final Inspection" or "Plumbing Rough-In Checklist"). Maximum 512 characters.

Description

No

Optionally, provide additional context about when or how this checklist should be used. Maximum 512 characters.

Step 3: Add Checklist Items

Below the title and description, you will see the Checklist Items section. This is where you define the individual tasks or verification points that make up the checklist.

To add an item:

  1. Type the item title in the text field (e.g., "Verify all connections are sealed").
  2. Press Enter or click the checkmark button to add the item.
  3. Repeat for each item you want to include.

For each item, you can also check the "Description Required" checkbox. When this is checked, the person completing the job will be required to write a description for that item before the job can be marked as complete. This is useful for items that need documented evidence or notes (e.g., "Record the serial number of the installed unit").

After adding items, they will appear in a list below. You can:

  • Edit an item by clicking the pencil icon next to it.
  • Remove an item by clicking the trash icon next to it.

Note: You must add at least one item before you can save the checklist. The "Add New Checklist" button will remain disabled until at least one item has been added.

Step 4: Save the Checklist

Once you have entered the title and added your items, click the "Add New Checklist" button at the bottom of the panel. You will see a brief "Creating Checklist..." overlay, and once the checklist is created successfully, a confirmation message will appear: "Checklist Successfully Created"

.

The new checklist will now appear in your Checklists List View and be available for assignment to jobs and estimates.

2. Viewing and Managing Checklist Items

To view or edit the items in an existing checklist, click on any checklist row in the Checklists List View. This will open the Checklist Detail page.

What You See

The detail page shows:

  • The checklist name at the top, with a pencil icon to edit the name and description.
  • A status badge showing whether the checklist is "Active" or "Archived".
  • A "New Item" button to add more items.
  • A table of all checklist items with the following columns:

Column

What It Means

Drag Handle

Grip dots on the left side of each row. Click and drag to reorder items (see below).

Title

The name/description of the checklist item.

Description Required

A toggle indicator. Blue (filled) means a description is required when completing this item. Gray (hollow) means it is optional. You can click this toggle to change the setting directly from the table.

Edit

Pencil icon to open the item for editing.

Delete

Trash icon to remove the item from the checklist.

Adding a New Item

  1. Click the "New Item" button at the top of the detail page.

  1. A dialog will appear with two fields:
  • Title (required) — Enter the item name. Maximum 512 characters.
  • Description Required (optional) — Check this box if a description must be provided when completing this item.

  1. Click "Add Item" to save, or "Cancel" to go back.

 

  1. A success message will appear: "Checklist Entry Successfully Created".

Editing an Item

  1. Click the pencil icon next to the item you want to edit.

  1. A dialog will appear pre-filled with the item's current title and description required setting.

  1. Make your changes and click "Apply Changes", or click "Cancel" to discard.

  1. A success message will appear: "Checklist Entry Successfully Updated".

Reordering Items (Drag and Drop)

You can change the order in which items appear by dragging and dropping them:

  1. Click and hold the drag handle (grip dots) on the left side of the item row.
  2. Drag the item up or down to the desired position.
  3. Release to drop it in place.

The new order is saved automatically. The items will appear in this order wherever the checklist is used (in jobs and when completing jobs).

Deleting an Item

  1. Click the trash icon next to the item you want to remove.

  1. A confirmation dialog will appear: "Are you sure you want to delete [Item Title]?"

  1. Click "Delete" to confirm, or "Cancel" to go back.
  2. A success message will appear: "Checklist Entry Successfully Deleted".

Note: You cannot delete the last item in a checklist. Every checklist must have at least one item. If you try to delete the last remaining item, the delete button will be disabled with a tooltip: "Cannot delete last item."

3. Editing a Checklist's Name and Description

To change the name or description of a checklist (not its items):

  1. Open the checklist detail page by clicking on it in the Checklists List View.

  1. Click the pencil icon next to the checklist name at the top of the page.
  2. A panel will slide open with two editable fields:
  • Checklist Title (required) — Maximum 512 characters.
  • Description (optional) — Maximum 512 characters.
  1. Make your changes and click "Apply Changes".
  2. A success message will appear: "Checklist Successfully Updated".

Tip: You can also access the edit panel by clicking the pencil icon in the Actions column on the Checklists List View.

4. Duplicating a Checklist

If you want to create a new checklist that is similar to an existing one, you can duplicate it instead of starting from scratch:

  1. On the Checklists List View (Active tab), find the checklist you want to duplicate.

  1. Click the duplicate icon (copy icon) in the Options column.
  2. A confirmation dialog will appear: "Are you sure you want to duplicate [Checklist Name]?"
  3. Click "Duplicate" to confirm.
  4. A success message will appear: "Checklist Successfully Duplicated".

The duplicated checklist will appear in the list as a new, independent copy. You can then edit its name, description, and items without affecting the original.

5. Archiving Checklists

Archiving is a safe way to disable a checklist without permanently deleting it. Archived checklists:

  • Do not appear in the dropdown when creating or editing jobs and estimates.
  • Are preserved in historical records — any existing jobs that have this checklist assigned will continue to work normally.
  • Can be restored at any time if you need the checklist again.

How to Archive a Checklist

  1. On the Checklists List View (Active tab), locate the checklist you want to archive.
  2. Click the archive icon in the Options column.

  1. A confirmation dialog will appear: "Are you sure you want to archive [Checklist Name]?"
  2. Click "Archive" to confirm, or "Cancel" to go back.
  3. You will see an "Updating Checklist..." overlay, followed by a success message: "Checklist Successfully Archived".

The checklist will be removed from the Active tab and moved to the Archived tab.

5.1 Restoring an Archived Checklist

If you need to bring an archived checklist back into active use:

  1. Switch to the Archived tab on the Checklists List View.

  1. Find the checklist you want to restore.
  2. Click the restore icon (undo icon) in the Options column.
  3. A confirmation popover will appear: "Do you want to restore this checklist?"
  4. Click "Restore" to confirm.
  5. A success message will appear: "Checklist Successfully Restored".

The checklist will be moved back to the Active tab and will once again appear in the dropdown when creating or editing jobs.

5.2 Permanently Deleting a Checklist

Permanent deletion is only available for archived checklists and only when the checklist is not in use (not assigned to any job).

  1. Switch to the Archived tab on the Checklists List View.

  1. Find the checklist you want to permanently delete.
  2. Click the delete icon (trash icon) in the Options column.
  • If the delete icon is grayed out (disabled), it means the checklist is still assigned to at least one job. You will see a tooltip: "Checklist cannot be deleted while in use."
  1. A confirmation dialog will appear: "Are you sure you want to delete [Checklist Name]?"
  2. Click "Delete" to confirm, or "Cancel" to go back.
  3. You will see a "Deleting Checklist..." overlay, followed by a success message: "Checklist Successfully Deleted".

Warning: Permanent deletion cannot be undone. Once a checklist is deleted, it is removed from the system permanently. If you are unsure, keep the checklist archived instead.

6. How Checklists Are Used Across the Platform

Checklists are primarily used in Jobs and Estimates. When you assign a checklist to a job, the job gets its own independent copy of the checklist — so any changes made to items during the job (adding entries, filling in descriptions) do not affect the original template.

Assigning a Checklist to a Job

When creating or editing a job, you will see a "Checklist (Optional)" dropdown in the Job Details step. This dropdown lists all your active checklists (archived checklists are not shown). Select a checklist to assign it to the job.

Once a checklist is assigned to a job:

  • A "Checklist" tab appears on the job detail page.

  • The Checklist tab shows all the items from the template, displayed in order.
  • You can expand each item by clicking the chevron to enter a description or notes for that item.
  • If an item has "Description Required" enabled, it will be marked with an asterisk (*), and a description must be provided before the job can be completed.
  • You can also add new items directly to the job's checklist — these additions only affect the job, not the original template.
  • You can delete items from the job's checklist — again, this only affects the job copy.

Completing a Job with a Checklist

When you attempt to mark a job as complete, and the job has a checklist assigned:

  1. A completion modal will appear showing all checklist items with checkboxes.
  2. You must check every item to confirm it has been completed or verified.
  3. For items marked as "Description Required", a description text field will appear when you check the item. You must fill in the description before proceeding.
  4. The "Complete Job" button will remain disabled until all items are checked and all required descriptions are filled in.

This ensures that no job is marked as complete without first verifying all items on the checklist.

Note: If a job does not have a checklist assigned, the completion modal will simply ask: "Are you sure you want to complete this job?" without any checklist review.

Assigning a Checklist to an Estimate

Similar to jobs, you can assign a checklist to an estimate using the "Checklist (Optional)" dropdown on the Estimate Details step. When the estimate is later converted to a job, the checklist assignment carries over.

Understanding Checklist Items

Each item in a checklist is simple by design. An item has just two properties:

Property

What It Means

Title

A short description of what needs to be checked or verified (e.g., "Test water pressure at all fixtures"). Maximum 512 characters.

Description Required

When enabled, the person completing the job must write a description or notes for this item (e.g., "Record the pressure reading in PSI"). When disabled, the item only needs to be checked off.

This simplicity makes checklists easy to create and fast to complete in the field, while the "Description Required" option provides accountability for items that need documented verification.
