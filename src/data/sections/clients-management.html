Clients Management

Overview

The Clients Management section is where you manage all your client accounts, their properties, contact information, and billing details. A client represents a customer you do business with — they can be residential homeowners or commercial businesses. Each client can have multiple properties, and all appointments, estimates, jobs, and invoices are linked to client accounts.

From this section you can:

  • Add new clients with a multi-step wizard covering personal details, property, billing, and attachments.
  • Edit existing clients to update their contact information, properties, or billing addresses.
  • Archive clients you no longer work with (without losing historical data).
  • Restore archived clients if you need them again.
  • Search and filter by keyword, city, state, tags, and client type.
  • Import clients in bulk via CSV files.
  • Export your client list to CSV for external use.
  • Manage properties — add, edit, or delete properties for each client.
  • View related records — see all appointments, estimates, jobs, and invoices linked to a client.
  • Communicate with clients through the built-in messaging system.
  • Manage attachments for both clients and their properties.

Important: Clients cannot be permanently deleted — they can only be archived. Archiving preserves all historical data (appointments, estimates, jobs, invoices) while hiding the client from active views. You can restore an archived client at any time.

Clients List View

The Clients List View is the main view for managing all your clients. To access it, navigate to Menu > Clients.

When you open the Clients List View, you will see a table listing your clients. Each row represents one client account and displays the following information:

Column

What It Means

User

The client's avatar, primary user name and company name as a subtitle if available. If the client was created via Quick Add and has incomplete information, a red warning icon appears with a tooltip: "Created via Quick Add - details incomplete".

Type

The client's primary property type, shown with a colored dot: blue dot + "Residential" for residential clients, or orange dot + "Business" for commercial clients.

Phone

The primary user's phone number, formatted for display.

Email

The primary user's email address or the client account email. Truncated with an ellipsis if too long, with the full email visible on hover.

Address

The primary property's address (street, city, state, ZIP). Falls back to the billing address if no property exists. If the client has multiple properties, a tag showing +N appears (e.g., "+2") with a tooltip showing the breakdown by type (e.g., "1 Residential, 2 Business").

Tags

Color-coded tag chips assigned to the client. Shows "—" if no tags are assigned.

Actions

An edit button (pencil icon) to open the client detail page, and a three-dot menu with additional actions.

Tabs

At the top of the list you will see four tabs that filter clients by type:

  • All — Shows all active clients, regardless of type. This is the default view.
  • Residential — Shows only active clients whose primary property type is Residential.
  • Business — Shows only active clients whose primary property type is Commercial/Business.
  • Archived — Shows clients that have been archived.

You can switch between these tabs at any time. The table updates to reflect the selected filter.

Sorting the List

You can sort the clients list by clicking on the column headers for User (sorts by company name) or Type (sorts by account type). Click once to sort in ascending order and click again to sort in descending order.

Searching and Filtering

The Clients List View provides several tools to find clients quickly:

  • Search bar — Type a keyword to search across client names, email addresses, company names, and more. Results update in real time.
  • Cities filter — A multi-select dropdown populated with all cities from your client properties. Select one or more cities to show only clients in those locations.
  • States filter — A multi-select dropdown populated with all states. Select one or more states to narrow the results.
  • Tags filter — A multi-select dropdown with all tags used across clients. Select tags to show only clients with those tags.

The filters can be combined — selecting "New York" in States AND "HVAC" in Tags will show only clients in New York who also have the "HVAC" tag.

Row Actions Menu

Clicking the three-dot menu (⋮) on any row reveals the following actions:

  • Start a Conversation — Opens the Inbox with this client pre-selected. If a direct chat already exists, it opens the existing conversation. If not, a new chat is created.
  • Attachments — Opens the Client Attachments panel to manage files associated with this client.
  • Archive (active clients only) — Opens a confirmation dialog: "Are you sure you want to archive [client name]?". Archiving removes the client from active tabs but preserves all data.
  • Restore (archived clients only) — Opens a confirmation dialog: "Are you sure you want to restore [client name]?". Restoring moves the client back to the active tabs.

You can also click anywhere on a row to navigate to that client's Detail Page.

1. Adding a New Client

To add a new client to your account, follow these steps:

Step 1: Open the New Client Form

Click the "+ New Client" button (blue) located in the top-right corner of the Clients List View. A drawer panel will slide open from the right side with a five-step wizard.



Tip: If the list is empty, you will also see a "New Client" button in the center of the page.

Step 2: General Information (Step 1 of 5)

First, select the client type by clicking one of the two buttons at the top:

  • Residential (blue, person icon) — For individual homeowners.
  • Business (blue, briefcase icon) — For companies and commercial clients.

Step 5: Attachments (Step 4 of 5)

This step allows you to upload files for the client and/or the property. Two tabs are available:

  • Client Attachments — Files associated with the client account (contracts, agreements, IDs, etc.).
  • Property Attachments — Files associated with the property (photos, floor plans, permits, etc.).

Each tab has an independent upload area where you can:

  1. Drag and drop files into the dashed border area, or click to browse.
  2. Each file shows a progress bar during upload.
  3. Uploaded files display the file name, size, and a delete button.
  4. If an upload fails, a "Retry" button appears.

Note: Attachments are optional. You can skip this step and add attachments later from the Client Detail Page.

Click "Next" to proceed.

Step 6: Review (Step 5 of 5)

The review step shows a summary of all the information you entered:

  • Property Image — A large preview of the uploaded property photo.
  • Client Information — Name, type (with colored dot), phone, email, company name, website, and full property address. State codes are displayed as full names (e.g., "NY" shows as "New York").

Review the information carefully, then click "Create Client" to submit.

Tip: You can click any step indicator at the top of the wizard to jump back and make changes before submitting. In create mode, you can only go back to previously completed steps.

2. Client Detail Page

The Client Detail Page shows all information about a single client and their related records. To access it, click on any client's row or edit icon in the Clients List View.

Header Area

The header displays:

  • Back button — Returns to the Clients List View.
  • Client name — The company name (or "Unknown Client" if none).
  • Status badge — "Active" (green) if the client is not archived, or "Archived" (gray) if archived.
  • Start a Conversation button — Opens the Inbox with this client pre-selected for messaging.
  • Three-dot menu — Contains: Attachments, Archive (or Restore if archived).

Sidebar

The right sidebar displays:

  • Company name and property type tags (Residential/Business colored dots).
  • Phone — Primary user's phone number.
  • Email Address — Client account email.
  • Billing Address — Full formatted billing address (or "—" if none).
  • Tags — All assigned tags as colored chips.
  • Edit button — Opens the Edit Client modal with all data pre-filled.
  • Notes section — Internal notes with the ability to create, edit, and attach files.

2.1 Properties Tab

The Properties tab shows all properties associated with this client. A count badge in the tab header shows the number of active properties.

At the top, a "+ New Property" button allows adding additional properties.

Each property is displayed as a card showing:

  • Property type badge — "Residential" or "Commercial".
  • Property image — The uploaded photo or a placeholder.
  • Full address — Street, city, state, and ZIP.
  • Quick Add indicator — A warning icon if the property was created via Quick Add and has incomplete data.
  • "View on Map" link — Opens the property location in the Map view.
  • Action buttons — Attachments (paperclip icon), Edit (pencil icon), Delete (trash icon).

If no properties exist, the message "No Properties" is displayed.

2.2 Appointments Tab

Shows all appointments associated with this client. A count badge in the tab header shows the number of appointments (with a loading skeleton while fetching).

Each appointment is listed and clickable — clicking navigates to the Appointment Detail Page. If the client has only one active property, it is automatically pre-selected when creating a new appointment.

2.3 Estimates Tab

Shows all estimates associated with this client. A count badge shows the estimate count.

A "+ New Estimate" button at the top allows creating a new estimate with the client pre-selected. Clicking any estimate navigates to the Estimate Detail View.

2.4 Jobs & Invoices Tab

Shows all jobs and invoices associated with this client. A count badge shows the job count.

Two buttons at the top: "+ New Job" and "+ New Invoice" — both pre-fill the client. Clicking any job or invoice navigates to the respective Detail View.

3. Editing a Client

To update information for an existing client:

Step 1: Open the Client for Editing

You can edit a client from two places:

  • From the Client Detail Page — Click the "Edit" button in the sidebar.
  • From the Clients List View — Click the edit icon (pencil) in the Actions column.

The same five-step drawer opens, pre-filled with the client's current information. The title changes to "Edit Client".

Step 2: Make Your Changes

In edit mode, all five steps are accessible — you can click any step indicator to jump directly to that section without needing to complete previous steps sequentially.

You can modify:

  • General — Name, phone, email, company name, website, tags. Note: the client type toggle is not shown in edit mode.
  • Property — Property type, address, image, timezone, notes.
  • Billing — Billing address fields, "same as property" checkbox.
  • Attachments — Upload new or remove existing files.
  • Review — Verify all changes before saving.

Note: In edit mode, the email uniqueness check is not performed (since the client already owns that email). The client field in the edit panel displays the client's current information, which is read-only for the client account association but editable for the user details.

Step 3: Save Your Changes

Navigate to the Review step and click "Save Changes". The system updates the client account, user information, and property in separate operations. On success, a toast appears: "Client successfully updated".

Click "Cancel" at any step to close the drawer without saving.

4. Archiving Clients

Archiving is a safe way to deactivate a client without losing any historical data. Archived clients:

  • Do not appear in the All, Residential, or Business tabs — they only appear in the Archived tab.
  • Are preserved in historical records — existing appointments, estimates, jobs, and invoices remain intact.
  • Can be restored at any time.

How to Archive a Client

  1. On the Clients List View, find the client you want to archive.
  2. Click the three-dot menu (⋮) in the Actions column.
  3. Select "Archive".
  4. A confirmation dialog appears: "Are you sure you want to archive [client name]?"
  5. Click "Archive" (red) to confirm, or "Cancel" to go back.
  6. On success, a toast appears: "Client archived successfully".

You can also archive a client from the Client Detail Page using the three-dot menu in the header.

4.1 Restoring an Archived Client

To bring an archived client back into active use:

  1. Switch to the Archived tab on the Clients List View.
  2. Find the client you want to restore.
  3. Click the three-dot menu (⋮) and select "Restore".
  4. A confirmation dialog appears: "Are you sure you want to restore [client name]?"
  5. Click "Restore" to confirm.
  6. On success, a toast appears: "Client restored successfully".

The client will be moved back to the appropriate active tab (All, Residential, or Business).

5. Managing Properties

Each client can have multiple properties. Properties are managed from the Properties tab on the Client Detail Page.

5.1 Adding a New Property

  1. On the Client Detail Page, go to the Properties tab.
  2. Click the "+ New Property" button.
  3. A modal opens with the property form fields (same fields as Step 2 of the client wizard): property type, property image, address, city, state, ZIP, timezone, and address notes.
  4. Fill in the required fields and click "Save" to create the property.

5.2 Editing a Property

  1. On the Properties tab, click the edit icon (pencil) on the property card you want to modify.
  2. The same property form opens, pre-filled with the property's current data.
  3. Make your changes and click "Save".

5.3 Deleting a Property

  1. On the Properties tab, click the delete icon (trash) on the property card.
  2. A confirmation dialog appears: "Are you sure you want to delete this property?"
  3. Click "Delete" (red) to confirm


Warning: Deleting a property is permanent and cannot be undone. Consider whether any active appointments, estimates, or jobs reference this property before deleting.

5.4 Property Attachments

Each property has its own set of attachments, managed separately from client-level attachments. Click the paperclip icon on any property card to open its Attachments panel.

5.5 Viewing a Property on the Map

Click the "View on Map" link on any property card to open the Map view centered on that property's location.

6. Importing Clients via CSV[a][b]

You can import clients in bulk using a CSV file. To access the import feature, click the "Import CSV" button on the Clients List View.

Note: The Import CSV button may not be visible if it has been disabled by your system administrator.

Step 1: Upload Your CSV File

Drag and drop a CSV file into the upload area, or click to browse your computer. The system validates the file format before proceeding.

If you don't have a CSV file ready, click "Download Sample CSV" to get a template with example data and the expected column headers.

Step 2: Map Columns

The system shows your CSV headers and asks you to map each column to the corresponding client field. The following fields are available for mapping:

Required:

  • Company Name
  • Email

Optional:

  • Website
  • Property 1 and Property 2: Name, Contact Name, Contact Email, Contact Phone, Type, Address Line 1, Address Line 2, City, State, ZIP, Country
  • Billing: Company Name, Contact Name, Address Line 1, Address Line 2, City, State, ZIP, Country
  • Tags (comma-separated)
  • Timezone
  • Is Archived (Yes/No)

Step 3: Preview and Validate

A data grid shows the mapped data so you can review before importing. The system validates:

  • Required fields have values.
  • Email addresses are in valid format.
  • Property types are either "RESIDENTIAL" or "COMMERCIAL".
  • State codes are valid.

Step 4: Import

Click "Import" to begin the import process. The system shows progress and, on completion, displays the number of successfully imported clients and any errors. A toast message summarizes the result.

7. Exporting Clients to CSV[c]

To download your client list as a spreadsheet file:

  1. Click the "Export CSV" button on the Clients List View.
  2. The system generates a CSV file containing clients matching your current filters (tab, search, city, state, tags).
  3. The file downloads automatically with a name like clients_2026-02-19_14-30.csv.

On success, a toast appears: "Clients exported successfully".

Tip: Use the search and filter tools to narrow down the export. For example, switch to the "Business" tab before exporting to get only commercial clients.

Note: The Export CSV button may not be visible if it has been disabled by your system administrator.

8. Client Attachments

Attachments allow you to associate files with a client account. These are separate from property-level attachments.

Opening the Attachments Panel

There are three ways to manage client attachments:

  1. From the Clients List View — Click the three-dot menu and select "Attachments".
  2. From the Client Detail Page — Click the three-dot menu in the header and select "Attachments".
  3. During client creation — Attachments are part of Step 4 of the five-step wizard.

Uploading Files

  1. Drag and drop files into the upload area, or click to browse.
  2. A progress bar appears for each file during upload.
  3. Uploaded files display the file name, size, and type icon.

Deleting Files

Click the trash icon next to any file to delete it. Deletion is permanent.

9. Quick Add Feature

Quick Add allows creating clients with minimal information for rapid data entry. When a client is created via Quick Add (from another section such as Estimates or Jobs), the record is flagged as incomplete.

Identifying Quick Add Clients

Quick Add clients are identified by a red warning triangle icon next to their name in the Clients List View. A tooltip reads: "Created via Quick Add - details incomplete".[d][e][f]

Completing Quick Add Clients

  1. Click the warning icon or the "Complete Info" button in the tooltip.
  2. The Edit Client modal opens with whatever data was entered during Quick Add.
  3. Fill in the remaining required fields.
  4. Click "Save Changes" to complete the client.

Once saved, the Quick Add flag is removed and the warning icon disappears.

10. Searching and Filtering Clients

10.1 Quick Search

The search bar at the top of the Clients List View searches across client names, company names, email addresses, and other fields. Type your search term and results update automatically.

10.2 Advanced Filtering

Three multi-select filter dropdowns are available:

Filter

What It Does

Cities

Shows only clients with properties in the selected cities.

States

Shows only clients with properties in the selected states.

Tags

Shows only clients that have the selected tags assigned.

10.3 Tab Filtering

The four tabs (All, Residential, Business, Archived) provide quick filtering by client type and status. Tab filtering is combined with search and advanced filters.

Tip: You can combine all filters for maximum precision. For example, search for "Smith" on the "Residential" tab while filtering by "Texas" in States and "Premium" in Tags to find all residential clients named Smith in Texas with the Premium tag.
