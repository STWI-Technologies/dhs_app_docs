Estimates Management

Overview

The Estimates Management section is where you create, manage, and track service estimates for your clients. An estimate is a proposed scope of work — including services, products, and pricing — that you prepare for a client before starting a job. Once the client reviews and approves the estimate, you can convert it directly into a job.

From this section you can:

  • Create new estimates with client, property, crew, and service/product line items.
  • Edit existing estimates to update details or line items.
  • Submit estimates to clients via email and/or SMS for review.
  • Track status through the estimate lifecycle (Draft, Ready for Review, Changes Requested, Approved).
  • Convert to Job approved estimates into scheduled jobs with a single action.
  • Duplicate estimates to quickly create similar proposals.
  • Delete estimates that are no longer needed.
  • Download PDF copies of estimates for offline sharing or records.
  • Manage billing with alternate billing addresses and "Bill To" client assignments.
  • Communicate with clients through built-in chat, notes, and attachments.

Important: Estimates in Ready for Review or Approved status are read-only — you cannot edit service items, products, or billing details. To make changes, return the estimate to Draft or Changes Requested status first. The recurring schedule tab remains editable regardless of status.

Estimates List View

The Estimates List View is the main view for managing all your estimates. To access it, navigate to Menu > Estimates.

When you open the Estimates List View, you will see a table listing all your estimates. Each row represents one estimate and displays the following information:

Column

What It Means

Name

The title of the estimate. Click the name to navigate to the Estimate Detail View. If the estimate was created via Quick Add (incomplete), a warning icon will appear next to the name indicating that additional information is needed.

Client

The client associated with the estimate, shown with their avatar (or initials) and company name. If no client is assigned, it displays "No Client".

Total

The total estimated amount in dollars (e.g., $1,250.00). Amounts are stored in cents and automatically formatted for display.

Status

The current status of the estimate, shown as a color-coded tag. If the estimate has a chat conversation, a chat icon will also appear. A blue chat icon with a red badge indicates unread messages.

Date Created

The date the estimate was created, formatted as Mon DD, YYYY (e.g., "Feb 10, 2026").

Actions

Quick-action buttons: edit (pencil icon to open detail view), duplicate (copy icon), and delete (trash icon).

Every estimate has one of four statuses. These statuses help you track where each estimate is in the approval process:

Sorting the List

You can sort the estimates list by clicking on the column headers for Name, Total, Status, or Date Created. Click once to sort in ascending order and click again to sort in descending order.

Searching

Use the search bar at the top of the Estimates List View to search for estimates by keyword. The search looks across estimate names and client names. Simply type your search term and the results will update automatically. If no estimates match your keyword, you will see the message: "No estimates found matching your keyword".

Filtering by Status

Click the "Status" filter button to open a popover with status checkboxes. You can select one or more statuses to show only estimates matching those statuses:

  • Draft
  • Approved
  • Ready for Review
  • Changes Requested

When filters are active, a badge with a number appears on the filter button indicating how many status filters are applied. To remove all filters, open the popover and click "Reset filter".

Pagination

The Estimates List View supports server-side pagination. You can change the number of estimates displayed per page and navigate between pages using the pagination controls at the bottom of the table.

Row Actions

Each row in the estimates table has three quick-action buttons in the Actions column:

  • Edit (pencil icon) — Opens the Estimate Detail View for the selected estimate.
  • Duplicate (copy icon) — Creates an exact copy of the estimate. A loading overlay displays "Duplicating Estimate..." while the operation completes.
  • Delete (trash icon) — Opens a confirmation dialog to permanently delete the estimate (see Section 4).

You can also click anywhere on a row to navigate to that estimate's Detail View.

Chat Icon in Status Column

If an estimate has an associated chat conversation, a chat icon appears next to the status tag. The icon behavior is:

  • Gray background — A chat exists but there are no unread messages. Clicking navigates to the Inbox with the conversation open.
  • Blue background with red badge — There are unread messages. The badge shows the unread count (up to "99+"). Clicking navigates to the Inbox.

1. Creating a New Estimate

To create a new estimate, follow these steps:

Step 1: Open the New Estimate Form

Click the "+ New Estimate" button (blue) located in the top-right corner of the Estimates List View. A drawer panel will slide open from the right side of the screen with a two-step wizard.

Tip: If the list is empty (no estimates have been created yet), you will also see a "New Estimate" button in the center of the page.

Step 2: Fill In the Estimate Details (Step 1 of 2)

The first step of the wizard is titled "Estimate Details" and contains the following fields:

Field

Required?

Description

Title

Yes

Enter a descriptive name for the estimate (e.g., "Kitchen Renovation Estimate" or "Annual HVAC Maintenance Proposal").

Client

Yes

Select the client this estimate is for from the dropdown. You can search by company name or email. If the client does not exist yet, click the "Quick Add" option to create a new client without leaving the form. The new client will be automatically selected after creation.

Property

Yes

Select the property where the work will be performed. Properties are filtered based on the selected client. If no client is selected, this field displays "Select client first" and is disabled. You can also Quick Add a new property.

Crew

Yes

Select the crew that will perform the work. You can also Quick Add a new crew from this dropdown.

Checklist

No

Optionally select a checklist to associate with this estimate. Checklists help ensure all required steps are completed when the estimate is converted to a job.

After filling in all required fields, click "Next Step" to proceed. The system will validate your entries — if any required field is missing, you will see an error message next to that field.

Note: You can click on the step indicators at the top of the wizard to navigate between steps. Step 2 is only accessible once Step 1 passes validation.

Step 3: Add Service Items (Step 2 of 2)

The second step is titled "Service Items" and allows you to add services and products to the estimate. This step is optional — you can add items now or later from the Estimate Detail View.

From this step you can:

  • Add Services — Select from your existing service catalog. For each service, specify the hours and hourly rate. The total is calculated automatically.
  • Add Products — Select from your product catalog. For each product, specify the quantity and unit price. The total is calculated automatically.

Step 4: Save the Estimate

You have two options for saving:

  • "Add Estimate" (blue button) — Saves the estimate with all selected service items and products. A loading overlay displays "Creating Estimate..." while the operation completes.
  • "Save as Draft" (text button) — Saves the estimate without adding any service items from Step 2. This is useful if you want to fill in the line items later.

Both options create the estimate in Draft status. The new estimate will appear in your Estimates List View.

Tip: To go back and modify the estimate details before saving, click the step indicator for "Estimate Details" or navigate back to the first step.

2. Estimate Detail View

The Estimate Detail View is where you manage all aspects of a single estimate. To access it, click on any estimate's name or edit icon in the Estimates List View.

The Detail View has two main areas:

  • Main content area (left) — Contains tabs for Overview and Recurring Schedule.
  • Client sidebar (right) — Shows client information, billing details, and communication tools (Chat, Notes, Attachments).

Header Area

The header displays:

  • Estimate name — The title of the estimate. If the estimate is part of a recurring series, a recurring badge icon appears next to the name.
  • Status dropdown — A color-coded tag showing the current status. Click it to change the status (see Section 5).
  • Action buttons — Vary depending on the current status (see below).
  • More actions menu (three-dot icon) — Contains additional actions: Duplicate, Download PDF, and Delete.
  • Back button — Returns to the Estimates List View. If you arrived from an Appointment Detail View, the back button returns you to that appointment's Related Records tab.

2.1 Overview Tab

The Overview tab is the default tab and displays all the estimate's details organized into sections:

Recurring Instance Banner

If the estimate is a recurring instance (part of a recurring series), a blue information banner appears at the top with two buttons:

  • "Edit This Estimate Instance" — Detaches this estimate from the series so changes apply only to this instance.
  • "Edit Series" — Navigates to the parent (template) estimate where you can modify the recurring schedule for all estimates in the series.

Overview Card

An "Edit" button (with pencil icon) appears in the top-right corner. Clicking it opens the Edit Estimate drawer (see Section 3). This button is disabled when the estimate is in a read-only status (Ready for Review or Approved).

The overview card displays a grid with the following fields:

Field

Description

Title

The name of the estimate.

Date Created

The date the estimate was created.

Property

The property name or address, with a Residential or Business tag if applicable.

Crew

The assigned crew name.

Checklist

The associated checklist name (if one was selected), shown as a gray tag. Only appears if a checklist is assigned.

Services Section

Lists all labor/service line items on the estimate. Each service shows the service name, description, hours, rate, and total. You can add, edit, or remove services when the estimate is not in a read-only status.

Products Section

2.2 Recurring Schedule Tab

The Recurring Schedule tab allows you to set up automatic recurring generation of estimates. This tab remains editable regardless of the estimate's status (even when the estimate is Approved or Ready for Review).

From this tab you can configure:

  • Recurrence frequency and pattern
  • Start and end dates for the series
  • Exceptions and modifications to individual instances

Client Information

The top section displays:

  • Client avatar (or initials)

  • Phone number

Two action buttons appear next to the client name:

  • Edit icon (pencil) — Opens the Billing Details modal to change the billing address.
  • Invoice icon — Opens the Bill To Client modal to assign a different client for billing. This button is disabled if a Bill To Client is already assigned (tooltip: "You can't add more than 1 bill to client").

  • Remove (trash icon) — Opens a confirmation dialog: "Are you sure you want to remove bill to client?" to remove the Bill To assignment.

3. Editing an Estimate

To update the details of an existing estimate:

Step 1: Open the Estimate for Editing

From the Estimate Detail View, click the "Edit" button (with pencil icon) in the Overview tab header. A drawer panel will slide open from the right side of the screen, pre-filled with the estimate's current information.

Note: The Edit button is disabled when the estimate is in Ready for Review or Approved status. To edit, first change the status back to Draft or Changes Requested using the status dropdown.

Step 2: Make Your Changes

The edit panel shows the following fields:

Field

Editable?

Description

Client

No (read-only)

Displays the client name with avatar. The client cannot be changed after the estimate is created.

Title

Yes (required)

Update the estimate name.

Property

Change the property. You can also Quick Add a new property from the dropdown.

Crew

Checklist

Click "Cancel" to close the panel without saving. Your changes will be discarded and the form will revert to the original values.

4. Deleting an Estimate

Estimates can be permanently deleted from two places:

From the Estimates List View

  1. Locate the estimate you want to delete in the list.
  2. Click the delete icon (trash icon) in the Actions column.
  3. A confirmation dialog will appear with the title "Delete Estimate" and the message: "Are you sure you want to delete this estimate?"
  4. Click "Delete" (red button) to confirm, or "Cancel" to go back.
  5. A loading overlay will display "Deleting Estimate..." while the operation completes.

From the Estimate Detail View

  1. Open the Estimate Detail View for the estimate you want to delete.
  2. Click the three-dot menu in the header.
  3. Select "Delete".
  4. The same confirmation dialog will appear.
  5. Click "Delete" to confirm.
  6. After deletion, you will be redirected to the Estimates List View. If you arrived from an Appointment Detail View, you will be redirected back to that appointment.

Warning: Permanent deletion cannot be undone. Once an estimate is deleted, all of its information, service items, products, notes, and attachments are removed from the system permanently.

7. Converting an Estimate to a Job

Once a client approves an estimate, you can convert it directly into a scheduled job.

Prerequisites

  • The estimate must be in Approved status.
  • The estimate must have at least one service item.

Step 1: Open the Convert to Job Wizard

From the Estimate Detail View (with Approved status), click the "Convert to Job" button. A drawer opens with a two-step wizard showing the job title and estimated duration at the top.

Step 2: Select a Start Date (Step 1 of 2)

A calendar is displayed showing available dates for the next 90 days. Dates are color-coded based on crew and client availability:

  • Available dates — Selectable and shown normally.
  • Unavailable dates — Grayed out and cannot be selected. These dates conflict with the assigned crew's schedule or the client's availability.

Select a start date and time, then click "Next Step".

Step 3: Select an End Date (Step 2 of 2)

Select the job's end date and time. The system validates that:

  • The end date/time is after the start date/time.
  • Both date and time fields are filled in.

Priority Flag

On both steps, a "Priority" checkbox is available at the bottom. Check this to mark the resulting job as high priority.

Step 4: Convert

Click "Convert to Job" to create the job. After conversion:

  • The system creates a new job with the estimate's client, property, crew, services, products, and billing details.
  • You are automatically navigated to the newly created Job Detail View.
  • The estimate now has a linked job_id. Returning to the estimate, the header shows a "View Job" button instead of "Convert to Job".

Note: If the estimate has already been converted to a job, opening the Convert to Job modal shows a success message: "This estimate has already been converted to job" with a "View Job" button to navigate to the existing job.

8. Duplicating an Estimate

To create a copy of an existing estimate:

From the Estimates List View

  1. Click the copy icon in the Actions column for the estimate you want to duplicate.
  2. A loading overlay displays "Duplicating Estimate..." while the copy is created.
  3. The new estimate appears in the list with the same details as the original.

From the Estimate Detail View

  1. Click the three-dot menu in the header.
  2. Select "Duplicate".
  3. The estimate is duplicated and the current view refreshes.

The duplicated estimate includes all the original's details (title, client, property, crew, services, products, billing) but is created as a new, independent estimate in Draft status.

9. Downloading an Estimate as PDF

To download a PDF copy of an estimate:

  1. Open the Estimate Detail View.
  2. Click the three-dot menu in the header.
  3. Select "Download PDF".
  4. The system generates and downloads a PDF file named [EstimateName].pdf.

The PDF can be shared with clients via other channels (print, email attachments, etc.) or kept for your records.

10. Searching and Filtering Estimates

10.1 Quick Search

The search bar at the top of the Estimates List View allows you to search for estimates by keyword. The search looks across estimate names and associated client names. Simply type your search term and the results will update automatically.

To clear the search, delete the text from the search bar or click the clear button.

10.2 Status Filter

Click the "Status" button to open the filter popover. Select one or more statuses to filter the list:

Filter Option

What It Shows

Draft

Estimates being prepared, not yet sent to client.

Approved

Estimates approved by the client, ready for conversion.

Ready for Review

Estimates sent to the client, awaiting their review.

Changes Requested

Estimates where the client has asked for modifications.

When filters are active, a badge with the number of active filters appears on the button. Click "Reset filter" in the popover to clear all filters.

Tip: You can combine the quick search and status filter for precise results. For example, search for "renovation" while filtering by "Approved" to find all approved renovation estimates.
